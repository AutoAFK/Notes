# Commands
| Command               | Description          |
| --------------------- | -------------------- |
| cd                    | change dir           |
| dir/ls                | list all files       |
| title                 | set title for prompt |
| del                   | delete file          |
| mkdir                 | make directory       |
| rmdir                 | remove directory     |
| type nul >> file_name | creates a file       |
